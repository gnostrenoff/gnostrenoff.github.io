Projet

Jnet Technologie est une entreprise spécialisé dans le developpement, la maintenance et le perfectionnement de systèmes d’exploitation pour micro-contrôleurs type carte à puce (smartcard) voués a différentes applications (banque, transport, santé, identité).

Mission

  • Dévéloppement de modules de communication supplémentaires sur la carte (tels que SPI, I2C et USB). Ajout d'une fonctionnalité "multi-interface" au systeme d'exploitation de la carte, pour que celui ci puisse supporter toutes les interfaces citées.
  • Optimisation de la sécurité et du protocol de communication et développement de tests de cryptographie (en majorité RSA et DES) en java.
  • Developpement d'une bibliothèque java sur une puce permettant de controler une LED, un capteur de toucher, et l'acces a un stockage USB.

Technologies et outils

  • Langages : C, Javacard, Java
  • Plateforme : Windows 7
  • IDE : Visual Studio, Keil 4
